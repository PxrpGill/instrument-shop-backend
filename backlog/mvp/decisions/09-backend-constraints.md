# 09 Backend Constraints

## Required Backend Constraints

Backend должен обеспечить:

- разделение публичного и внутреннего API;
- проверки permissions для внутренних endpoint'ов;
- валидацию публикации товара;
- фиксацию цены в заказе;
- тесты на visibility и permissions;
- миграции для новых полей и сущностей.
